## Get Blocks

Retrieve a list of the 15 most recent blocks.

<CodeGroup>
```bash cURL
curl "https://mempool.space/api/v1/blocks"
```

```javascript JavaScript
const response = await fetch('https://mempool.space/api/v1/blocks');
const blocks = await response.json();
console.log(blocks);
```

```python Python
import requests

response = requests.get('https://mempool.space/api/v1/blocks')
blocks = response.json()
print(blocks)
```
</CodeGroup>

<ResponseField name="blocks" type="Block[]">
  Array of block objects
  
  <Expandable title="Block Object">
    <ResponseField name="id" type="string">
      Block hash
    </ResponseField>
    
    <ResponseField name="height" type="number">
      Block height
    </ResponseField>
    
    <ResponseField name="version" type="number">
      Block version
    </ResponseField>
    
    <ResponseField name="timestamp" type="number">
      Unix timestamp of when the block was mined
    </ResponseField>
    
    <ResponseField name="bits" type="number">
      Compact representation of the target difficulty
    </ResponseField>
    
    <ResponseField name="nonce" type="number">
      Nonce value used for mining
    </ResponseField>
    
    <ResponseField name="difficulty" type="number">
      Block difficulty
    </ResponseField>
    
    <ResponseField name="merkle_root" type="string">
      Merkle root of all transactions in the block
    </ResponseField>
    
    <ResponseField name="tx_count" type="number">
      Number of transactions in the block
    </ResponseField>
    
    <ResponseField name="size" type="number">
      Block size in bytes
    </ResponseField>
    
    <ResponseField name="weight" type="number">
      Block weight (segwit)
    </ResponseField>
    
    <ResponseField name="previousblockhash" type="string">
      Hash of the previous block
    </ResponseField>
    
    <ResponseField name="mediantime" type="number">
      Median time of the last 11 blocks
    </ResponseField>
    
    <ResponseField name="stale" type="boolean">
      Whether this block is stale
    </ResponseField>
  </Expandable>
</ResponseField>

## Get Blocks at Height

Retrieve 15 blocks starting from a specific height.

<CodeGroup>
```bash cURL
curl "https://mempool.space/api/v1/blocks/800000"
```

```javascript JavaScript
const height = 800000;
const response = await fetch(`https://mempool.space/api/v1/blocks/${height}`);
const blocks = await response.json();
console.log(blocks);
```

```python Python
import requests

height = 800000
response = requests.get(f'https://mempool.space/api/v1/blocks/{height}')
blocks = response.json()
print(blocks)
```
</CodeGroup>

<ParamField path="height" type="number" required>
  Block height to start from
</ParamField>

<ResponseField name="blocks" type="Block[]">
  Array of 15 block objects starting from the specified height (see Block Object structure above)
</ResponseField>

## Get Block by Hash

Retrieve detailed information about a specific block by its hash.

<CodeGroup>
```bash cURL
curl "https://mempool.space/api/v1/block/000000000000000000076c036ff5119e5a5a74df77abf64203473364509f7732"
```

```javascript JavaScript
const blockHash = '000000000000000000076c036ff5119e5a5a74df77abf64203473364509f7732';
const response = await fetch(`https://mempool.space/api/v1/block/${blockHash}`);
const block = await response.json();
console.log(block);
```

```python Python
import requests

block_hash = '000000000000000000076c036ff5119e5a5a74df77abf64203473364509f7732'
response = requests.get(f'https://mempool.space/api/v1/block/{block_hash}')
block = response.json()
print(block)
```
</CodeGroup>

<ParamField path="hash" type="string" required>
  Block hash (64 character hex string)
</ParamField>

<ResponseField name="block" type="Block">
  Block object with complete details (see Block Object structure above)
</ResponseField>

```javascript JavaScript
const response = await fetch(`https://mempool.space/api/v1/block/${blockHash}/txs`);

```

```python Python

print(transactions)
```
</CodeGroup>

<ParamField path="hash" type="string" required>
  Block hash (64 character hex string)
</ParamField>

<ParamField path="index" type="number" optional>
  Starting index for pagination (defaults to 0). Returns 10 transactions per page.
</ParamField>

<ResponseField name="transactions" type="Transaction[]">
  Array of transaction objects in the block
  
  <Expandable title="Transaction Object">
    <ResponseField name="txid" type="string">
      Transaction ID
    </ResponseField>
    
    <ResponseField name="version" type="number">
      Transaction version
    </ResponseField>
    
    <ResponseField name="locktime" type="number">
      Transaction locktime
    </ResponseField>
    
    <ResponseField name="size" type="number">
      Transaction size in bytes
    </ResponseField>
    
    <ResponseField name="weight" type="number">
      Transaction weight (segwit)
    </ResponseField>
    
    <ResponseField name="fee" type="number">
      Transaction fee in satoshis
    </ResponseField>
    
    <ResponseField name="vin" type="Vin[]">
      Array of transaction inputs
    </ResponseField>
    
    <ResponseField name="vout" type="Vout[]">
      Array of transaction outputs
    </ResponseField>
    
    <ResponseField name="status" type="Status">
      Transaction confirmation status
    </ResponseField>
  </Expandable>
</ResponseField>
