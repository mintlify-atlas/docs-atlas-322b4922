## Overview

The Bitcoin Price API provides current Bitcoin exchange rates in multiple fiat currencies. Prices are updated regularly and cached for performance.

## Endpoint

### Get Current Prices

<ParamField path="GET" type="endpoint">
  `/api/v1/prices`
</ParamField>

Returns the latest Bitcoin exchange rates in multiple fiat currencies.

**Response Headers**

The response includes cache control headers:

- `Pragma: public`
- `Cache-control: public`
- `Expires`: Set based on price update frequency

## Usage Notes

### Caching

Price data is cached and updated at a configurable interval (default is multiple times per hour). The `Expires` header indicates when the next price update will occur.

### Currency Support

All major global currencies are supported. The availability of certain currencies may depend on the configured price data provider.

### Rate Limiting

This endpoint is designed for public access with appropriate caching headers. Clients should respect the cache headers to minimize redundant requests.

## Example Usage

### JavaScript/TypeScript

```typescript
async function getBitcoinPrices() {
  const response = await fetch('https://mempool.space/api/v1/prices');
  const prices = await response.json();
  
  console.log(`Bitcoin price in USD: $${prices.USD}`);
  console.log(`Bitcoin price in EUR: €${prices.EUR}`);
  
  return prices;
}
```

### Python

```python
import requests

def get_bitcoin_prices():
    response = requests.get('https://mempool.space/api/v1/prices')
    prices = response.json()
    
    print(f"Bitcoin price in USD: ${prices['USD']}")
    print(f"Bitcoin price in EUR: €{prices['EUR']}")
    
    return prices
```
